Security & Terms of Use

This tool is designed for LOCAL USE ONLY.
NEVER expose it on a network, VPN, reverse proxy, or the Internet.

What is protected

  • Server bound to 127.0.0.1 only
  • CORS restricted to localhost origins
  • PID allowlist: only scanned processes can be killed
  • Docker allowlist: only scanned containers can be stopped/restarted
  • No shell=True in system commands
  • HTML escaping against XSS injections
  • Docker processes filtered (no healthcheck pollution)
  • Zero external network calls (local icons, no CDN)

What is NOT protected

  • No authentication (unnecessary on localhost)
  • No TLS encryption (unnecessary on loopback)
  • No rate limiting
  • Process command lines may contain secrets
These choices are intentional. For a local tool on 127.0.0.1, these measures are sufficient.

NEVER do this

  • Change the bind from 127.0.0.1 to 0.0.0.0
  • Expose port 3999 via a reverse proxy
  • Use this tool on a shared server
  • Add "null" to CORS origins
